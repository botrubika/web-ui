## ✨ Features

- ✅ **Ping Testing** – Check if a host is reachable and measure response time
- 🔒 **Security Header Analysis** – Detect missing security headers (HSTS, CSP, X-Frame-Options, etc.)
- 🌐 **HTTP Status Monitoring** – Check status codes and response times
- 🔄 **DNS Resolution** – Resolve domain names to IP addresses
- 📊 **Security Score** – Calculate a security score from 0 to 5 based on headers
- 🎨 **Beautiful Web UI** – Modern, responsive, and user-friendly interface
- 🚀 **Lightweight & Fast** – Minimal dependencies, optimized for performance
- 🧪 **Fully Tested** – Includes unit tests with pytest
